The game was developped with Microsft Visual Studio 2010 and XNA 4.0.

To run the game on the PC, simply build and run the solution
Controls:
Left, right to move.
Space to interact.
Left Control to move objects.
Pause/Break to pause the game.
Enter to select from menu.


To run the game on Xbox 360, sign in with an Xbox Game Connect account.
Deploy the game from Visual Studio to the Xbox.
Controls:
Left, right analog stick to move.
A to interact.
Right trigger to move objects.
Start to pause the game, select from menu.
